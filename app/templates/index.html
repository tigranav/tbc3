Как работать с UI

  • 1
    Перейдите в раздел «Группы файлов книг». Там отображается актуальное содержимое таблицы из PostgreSQL.
  • 2
    Создайте группу и используйте её при добавлении типов файлов. Поле group_id в типах ссылается на books_files_groups.id.
  • 3
    Редактируйте и удаляйте записи прямо из таблиц. Кнопки действий обновляют данные через эндпоинты CRUD.
Все операции выполняются через существующие API /api/groups и /api/types с использованием подключения к PostgreSQL.
