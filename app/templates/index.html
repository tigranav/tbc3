Как работать с UI

  • 1
    Перейдите в раздел «Группы файлов книг». Там отображается актуальное содержимое таблицы из PostgreSQL.
  • 2
    Используйте форму для добавления. Введите идентификатор, название и комментарий — запись будет создана через API.
  • 3
    Редактируйте и удаляйте прямо из таблицы. Кнопки действий обновляют данные через эндпоинты CRUD.
Все операции выполняются через существующие API /api/groups и используют конфигурацию подключения к PostgreSQL.
